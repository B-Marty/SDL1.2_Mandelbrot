# SDL1.2_Mandelbrot
Mandelbrot Set in C SDL 1.2

### Dependencies
#### Linux Mint 19.3 Tricia
` sudo apt-get install libsdl1.2-dev `
#### Command
**A** : Auto zoom
**D** : Auto dezoom
**K** : Redraw
**H** : Log values
**O** : Decrement pixel size
**P** : Increment pixel size
**L** : Decrement max iteration number
**M** : Increment max iteration number
